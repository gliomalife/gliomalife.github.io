 ◦ Гранисетрон: из-за аллергии рекомендации Мацко на ХТ - Гранисетрон 1/2 млг/таблетки в сутки за 30-40 мин, при наличии аллергии заменить на Эменд День1-125 мг/сутки, Д2-Д3 80 мг/сутки.
